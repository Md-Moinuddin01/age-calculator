# age-calculator

A simple and accurate Age Calculator that calculates a person's exact age in years, months, and days based on their date of birth.

How It Works
User selects their Date of Birth.
The system calculates the difference between the current date and the birth date.

It displays the exact age in:
Years
Months
Days

The calculation uses built-in date functions for precision and avoids manual miscalculations.
